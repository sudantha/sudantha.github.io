# Why Modern CSS is Awesome

CSS has evolved significantly in recent years. With features like **Grid**, **Flexbox**, and **Custom Properties (Variables)**, we can build complex layouts with minimal effort.

## Key Features
- **CSS Grid**: Designing two-dimensional layouts.
- **Flexbox**: Perfect for one-dimensional alignment.
- **Variables**: Making themes easy to manage.

```css
:root {
  --primary-color: #6366f1;
}
```
